# 시스템 개요

## 프로젝트 비전

Claude는 AI 기반 서비스 플랫폼으로, 사용자와 관리자 모두를 위한 통합 솔루션을 제공합니다.

---

## 시스템 구성

### 사용자(User) 시스템

사용자 대상 서비스를 담당합니다.

| 기능 | 설명 | 상태 |
|------|------|------|
| 회원가입/로그인 | 이메일/소셜 로그인 지원 | 예정 |
| 프로필 관리 | 사용자 정보 수정 | 예정 |
| 서비스 이용 | 핵심 AI 서비스 제공 | 예정 |
| 결제 | 구독/일회성 결제 | 예정 |
| 이용 내역 | 사용 기록 조회 | 예정 |

### 관리자(Admin) 시스템

서비스 운영 및 관리를 담당합니다.

| 기능 | 설명 | 상태 |
|------|------|------|
| 대시보드 | 주요 지표 모니터링 | 예정 |
| 사용자 관리 | 회원 조회/관리 | 예정 |
| 콘텐츠 관리 | 서비스 콘텐츠 CRUD | 예정 |
| 결제 관리 | 결제 내역 조회/환불 | 예정 |
| 통계 | 사용 통계/리포트 | 예정 |

## 인증/로그인

### 인증 방식 (예정)

```
┌─────────────────────────────────────────────────────────┐
│                    인증 시스템                           │
├─────────────────────────────────────────────────────────┤
│  ┌───────────┐  ┌───────────┐  ┌───────────┐           │
│  │  이메일   │  │  Google   │  │  Kakao    │           │
│  │  로그인   │  │  OAuth    │  │  OAuth    │           │
│  └─────┬─────┘  └─────┬─────┘  └─────┬─────┘           │
│        │              │              │                  │
│        └──────────────┼──────────────┘                  │
│                       ▼                                 │
│              ┌───────────────┐                         │
│              │  JWT Token    │                         │
│              │  발급/검증    │                         │
│              └───────────────┘                         │
└─────────────────────────────────────────────────────────┘
```

### 인증 흐름

1. 사용자가 로그인 요청
2. 자격 증명 검증
3. Access Token + Refresh Token 발급
4. 클라이언트 토큰 저장
5. API 요청 시 토큰 포함
6. 토큰 만료 시 Refresh Token으로 갱신

## 결제 시스템

### 결제 유형 (예정)

| 유형 | 설명 | 결제 주기 |
|------|------|-----------|
| 무료 플랜 | 기본 기능 제공 | - |
| 월간 구독 | 프리미엄 기능 | 매월 |
| 연간 구독 | 프리미엄 기능 (할인) | 매년 |
| 일회성 결제 | 특정 서비스 구매 | 1회 |

### 결제 흐름

```
사용자 ──→ 플랜 선택 ──→ 결제 정보 입력 ──→ PG사 결제
                                              │
                                              ▼
결제 완료 ←── 권한 부여 ←── 결제 확인 ←── 웹훅 수신
```

### 연동 예정 PG사

> **주의**: 확정되지 않음. 프로젝트 요구사항에 따라 결정 필요.

- 토스페이먼츠
- 카카오페이
- 네이버페이
