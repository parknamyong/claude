## 2. 디자인 시스템

| 역할 | 색상 코드 | 용도 |

- 밝고 신뢰감 있는 톤

- 스크롤 탄력 (smooth scroll)
- 라인 아이콘 사용
- 이미지/영상은 placeholder로 대체

## 4. 필수 페이지

### 4.1 공개 페이지 (8개)

1. **홈** (`/`) - 메인 랜딩
2. **소개** (`/about`) - 교실 소개
3. **프로그램 상세** (`/programs/:slug`) - 3개 프로그램 각각
4. **수강신청** (`/enrollment`) - 신청 폼
5. **후기** (`/reviews`) - 수강 후기
6. **공지사항** (`/notices`) - 공지 목록
7. **갤러리** (`/gallery`) - 활동 사진
8. **오시는 길** (`/location`) - 위치 안내

### 4.2 학부모 대시보드

- 수강 정보
- 일정 확인
- 출석 현황
- 남은 회차
- 담당 강사
- 결제 내역
- 영수증 다운로드 (UI 시뮬레이션)
- 공지사항
- 프로필

### 4.3 어드민 대시보드

- 학생 관리
- 프로그램/반 관리
- 수강 신청 관리
- 출석 관리
- 결제/매출 관리
- 콘텐츠 관리
- 문자/이메일 발송 (UI 시뮬레이션)
- 엑셀 다운로드 (UI 시뮬레이션)
- 통계

## 5. 수강신청 플로우

### 5.1 결제 방식

| 방식 | 설명 |
|------|------|
| 즉시 결제 | 결제 UI 시뮬레이션 (실제 결제 X) |
| 상담 후 결제 | 결제 링크 발송 시뮬레이션 |

### 5.2 수집 정보

- **필수**: 학생 이름, 학년, 연락처, 보호자 연락처
- **선택**: 음악 경험, 희망 사항
- **동의**: 개인정보 수집/이용 동의

### 5.3 신청 완료

- 신청 완료 페이지 표시
- 관리자 알림 (UI 시뮬레이션)
- 문자 발송 (UI 시뮬레이션)

## 6. 기술 제약 (매우 중요)

> **이 제약은 반드시 준수해야 합니다.**

| 제약 | 설명 |
|------|------|
| **React 단일 앱** | Create React App / Vite 기반 SPA |
| **서버 없음** | 백엔드 서버 구현 금지 |
| **DB 없음** | 데이터베이스 사용 금지 |
| **API 없음** | 외부 API 호출 금지 |
| **Persistent Storage 금지** | 데이터 영속화 금지 |
| **localStorage 금지** | `localStorage` 사용 금지 |
| **sessionStorage 금지** | `sessionStorage` 사용 금지 |
| **상태 관리** | `React.useState`만 사용 |
| **결제** | 프로토타입(UI)만, 실제 결제 구현 금지 |
