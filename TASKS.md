## TOP 10 우선순위 작업

### 1. [ ] 프로젝트 초기 설정

**목적**: 개발 환경 표준화 및 협업 기반 마련

**범위**:
- 패키지 매니저 선정 (npm/yarn/pnpm)
- 프로젝트 구조 설계
- TypeScript/JavaScript 설정
- ESLint + Prettier 설정
- Git hooks (husky, lint-staged)

**검증 방법**:
- [ ] `npm install` 또는 해당 명령 정상 실행
- [ ] `npm run lint` 에러 없이 통과
- [ ] 커밋 시 pre-commit hook 동작 확인

### 3. [ ] 인증 시스템 설계 및 구현

**목적**: 사용자/관리자 로그인 기능 제공

**범위**:
- 회원가입 (이메일 인증)
- 로그인/로그아웃
- JWT 토큰 발급/갱신
- 소셜 로그인 (Google, Kakao)
- 비밀번호 재설정

**검증 방법**:
- [ ] 회원가입 → 로그인 → 로그아웃 플로우 테스트
- [ ] JWT 만료 시 Refresh 토큰으로 갱신 확인
- [ ] 소셜 로그인 연동 테스트
- [ ] 단위 테스트 커버리지 80% 이상
